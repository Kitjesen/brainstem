# 穹沛科技 - 专业设计系统

## 设计理念

### 核心原则
1. **柔和舒适**：避免纯白/纯黑，使用柔和的灰色调
2. **层次分明**：通过微妙的阴影和边框创建深度
3. **色彩克制**：使用柔和的品牌色，避免过于鲜艳
4. **简洁高效**：移除重复功能，保持界面清爽

## 配色方案

### 浅色模式（Light Mode）

#### 背景色
```dart
背景色：#F5F7FA  // 柔和的灰蓝色，不是纯白 #FFFFFF
卡片色：#FAFBFC  // 非常浅的灰色
```

**设计理由**：
- 纯白 (#FFFFFF) 在长时间使用时会造成眼睛疲劳
- 柔和的灰蓝色提供更舒适的视觉体验
- 卡片与背景的对比度适中，不会过于刺眼

#### 文字色
```dart
主文字：#1A1D1F  // 柔和的黑色，不是纯黑 #000000
次要文字：#6B7280  // 中灰色
```

**设计理由**：
- 纯黑 (#000000) 对比度过高，容易造成视觉疲劳
- 柔和的深灰色提供足够的可读性，同时更舒适

#### 边框和分隔线
```dart
边框：#E5E7EB  // 非常浅的灰色
分隔线：#E5E7EB
```

### 深色模式（Dark Mode）

#### 背景色
```dart
背景色：#0F1419  // 略亮于纯黑，带有蓝色调
卡片色：#1A1F2E  // 柔和的深蓝灰色
```

**设计理由**：
- 纯黑 (#000000) 在 OLED 屏幕上可能造成"烧屏"
- 略亮的背景色在 LCD 屏幕上更舒适
- 蓝灰色调营造专业、科技感

#### 文字色
```dart
主文字：#E8EAED  // 柔和的白色，不是纯白 #FFFFFF
次要文字：#9CA3AF  // 浅灰色
```

#### 边框和分隔线
```dart
边框：#374151  // 深灰色
分隔线：#374151
```

## 品牌色系统

### 柔和的品牌色（Muted Brand Colors）

```dart
紫色：#8B7FD6  // 原 #422AFB（过于鲜艳）
蓝色：#5B8DEF  // 原 #2563EB
青色：#4DB6AC  // 原 #0D9488
绿色：#66BB6A  // 原 #059669
橙色：#FF9F66  // 原 #EA580C
粉色：#EC7C9B  // 原 #DB2777
```

**设计理由**：
- 降低饱和度，避免过于刺眼
- 提高亮度，在深色背景上更易识别
- 保持色相，维持品牌识别度

### 状态色系统

```dart
成功/在线：#10B981  // 柔和的绿色
错误/离线：#EF4444  // 柔和的红色
警告：#F59E0B      // 柔和的橙色
信息：#06B6D4      // 柔和的青色
```

## 阴影系统

### 浅色模式阴影
```dart
卡片阴影：
  - 主阴影：rgba(0, 0, 0, 0.04), blur: 12px, offset: (0, 2px)
  - 次阴影：rgba(0, 0, 0, 0.02), blur: 4px, offset: (0, 1px)
```

**设计理由**：
- 双层阴影创造更自然的深度感
- 低透明度避免过重的阴影
- 小偏移量保持精致感

### 深色模式阴影
```dart
卡片阴影：
  - rgba(0, 0, 0, 0.3), blur: 8px, offset: (0, 2px)
```

**设计理由**：
- 深色模式需要更强的阴影来区分层次
- 单层阴影避免过于复杂

## 设计原则总结

### 1. 色彩克制
- ❌ 避免：纯白、纯黑、高饱和度颜色
- ✅ 使用：柔和的灰色调、降低饱和度的品牌色

### 2. 层次分明
- ❌ 避免：过重的阴影、过强的对比
- ✅ 使用：微妙的阴影、适度的对比、清晰的边框

### 3. 视觉舒适
- ❌ 避免：刺眼的白色、过暗的黑色
- ✅ 使用：柔和的背景色、舒适的文字色

### 4. 功能简化
- ❌ 避免：重复的按钮、冗余的功能
- ✅ 使用：集中的设置面板、清晰的层级

## 可访问性（Accessibility）

### 对比度
- 主文字对比度：≥ 7:1（AAA 级别）
- 次要文字对比度：≥ 4.5:1（AA 级别）
- 图标对比度：≥ 3:1（最低要求）

### 色盲友好
- 不仅依赖颜色传达信息
- 使用图标和文字辅助
- 状态指示器使用形状 + 颜色
