## 技术细节

### 透明度调整
```dart
// 深色模式：更透明
color: Colors.black.withValues(alpha: 0.5)  // 50%

// 浅色模式：更透明
color: Colors.white.withValues(alpha: 0.7)  // 70%

// 边框：更淡
border: Border.all(
  color: (isDark ? Colors.white : Colors.black)
         .withValues(alpha: 0.08),  // 8%
)

// 阴影：更柔和
BoxShadow(
  color: Colors.black.withValues(alpha: 0.15),  // 15%
  blurRadius: 30,
)
```

### 颜色系统
```dart
// 移除品牌色，使用主题色
cs.onSurface.withValues(alpha: 0.08)  // 背景
cs.onSurface.withValues(alpha: 0.5)   // 图标
cs.onSurface.withValues(alpha: 0.6)   // 按钮
cs.primary                             // 开关激活色
```

### 位置调整
```dart
// 从右下角改为左侧（侧边栏旁边）
Positioned(
  left: 88,   // 72px sidebar + 16px margin
  bottom: 16,
  child: ...
)

// 动画对齐点
alignment: Alignment.bottomLeft  // 从左下角缩放
```
